# 🚀 구글 애드센스 빠른 시작 가이드

## 💡 확인 방법 선택 가이드

구글 애드센스 승인을 위한 **소유권 확인 방법**을 선택해야 합니다:

### 🎯 **1. 애드센스 코드 스니펫** (가장 권장)
```html
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-YOUR_CLIENT_ID"
     crossorigin="anonymous"></script>
```

**✅ 장점:**
- 승인 후 바로 광고 표시 가능
- 자동 광고 기능 활용
- 실시간 성과 추적
- 구글이 자동으로 최적화

**❌ 단점:**
- 승인 전까지 빈 공간 표시
- 페이지 로드 시 스크립트 실행

**👍 추천 대상:**
- 빠른 수익화를 원하는 경우
- 자동 광고 기능을 사용하고 싶은 경우
- 기술적 설정을 최소화하고 싶은 경우

---

### 🏷️ **2. 메타 태그**
```html
<meta name="google-adsense-account" content="ca-pub-YOUR_CLIENT_ID">
```

**✅ 장점:**
- 가장 가벼운 방법 (성능 영향 최소)
- 승인 과정에서만 필요
- 페이지 로딩 속도에 영향 없음

**❌ 단점:**
- 승인 후 별도 광고 코드 설치 필요
- 자동 광고 기능 사용 불가
- 추가 설정 작업 필요

**👍 추천 대상:**
- 페이지 성능을 최우선으로 하는 경우
- 승인 받은 후 광고 위치를 세밀하게 제어하고 싶은 경우
- 트래픽이 매우 많은 사이트

---

### 🏆 **3. 둘 다 사용** (최고 승인률)
애드센스 코드 스니펫 + 메타 태그 + ads.txt

**✅ 장점:**
- 최고 승인 확률
- 모든 확인 방법 지원
- 광고 사기 방지 최대화

**❌ 단점:**
- 약간의 추가 설정 필요

**👍 추천 대상:**
- 확실한 승인을 원하는 경우
- 처음 애드센스를 신청하는 경우
- **우리 프로젝트에서 권장하는 방법**

## 🎯 **결론: 어떤 방법을 선택할까?**

### 🥉 **성능 최우선인 경우**

# 4번 질문에서 "2" 선택 (메타 태그)

## 🚀 **빠른 설정 (3분만에 완료)**

1. **애드센스 계정 생성**
   - [Google AdSense](https://www.google.com/adsense) 방문
   - 구글 계정으로 로그인
   - 웹사이트 URL 입력

2. **자동 설정 실행**
   ```bash
   npm run setup-adsense
   ```

3. **필요한 정보 입력**
   - 클라이언트 ID (ca-pub-으로 시작)
   - 광고 슬롯 ID (3개)
   - 광고 설정 선택

4. **웹사이트 배포**
   ```bash
   npm run build
   firebase deploy  # 또는 vercel deploy
   ```

5. **애드센스에서 사이트 추가**
   - 애드센스 계정에서 사이트 추가
   - 소유권 확인 완료
   - 승인 신청

## 📋 **승인 전 필수 체크리스트**

### 콘텐츠 준비사항
- [ ] 최소 15-20개의 고품질 기사
- [ ] 독창적이고 유용한 콘텐츠
- [ ] 정기적인 업데이트 (주 2-3회)
- [ ] 저작권 침해 없는 콘텐츠

### 페이지 준비사항
- [ ] 개인정보 처리방침 페이지
- [ ] 이용약관 페이지
- [ ] 연락처 정보 페이지
- [ ] 사이트맵 생성

### 기술적 준비사항
- [ ] SSL 인증서 설치 (HTTPS)
- [ ] 모바일 반응형 디자인
- [ ] 빠른 페이지 로딩 속도
- [ ] 404 에러 페이지 설정

### 트래픽 준비사항
- [ ] 일일 최소 100-500명 방문자
- [ ] 자연스러운 트래픽 소스
- [ ] 다양한 페이지 조회수

## ❓ **자주 묻는 질문**

### Q: 어떤 확인 방법이 가장 좋나요?
A: 처음 신청하는 경우 **"둘 다 사용"**을 추천합니다. 승인 확률이 가장 높습니다.

### Q: 승인까지 얼마나 걸리나요?
A: 보통 1-14일, 복잡한 경우 최대 30일 소요됩니다.

### Q: 거부되면 어떻게 하나요?
A: 거부 사유를 해결하고 최소 1주일 후 재신청할 수 있습니다.

### Q: 개발 환경에서 테스트할 수 있나요?
A: 네, 개발 환경에서는 테스트 광고가 표시됩니다.
